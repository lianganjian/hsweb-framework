# 功能,特性
- [x] 基于[r2dbc](https://github.com/r2dbc) ,[easy-orm](https://github.com/hs-web/hsweb-easy-orm/tree/4.0.x)的通用响应式CRUD
    - [x] H2,Mysql,SqlServer,PostgreSQL
- [x] 响应式r2dbc事务控制
- [x] 响应式权限控制,以及权限信息获取
    - [x] RBAC权限控制
    - [x] 数据权限控制
    - [ ] 双因子验证
- [x] 多维度权限管理功能
- [ ] 响应式动态数据源 
- [ ] 自定义拓展实体类
- [ ] 响应式缓存 
- [ ] 非响应式支持(mvc,jdbc)
- [ ] 内置业务功能
    - [x] 权限管理
        - [x] 权限设置
        - [x] 维度管理
        - [x] 权限分配
    - [ ] 文件上传
        - [x] 静态文件上传
        - [ ] 文件秒传
    - [ ] 数据字典
    - [ ] 组织架构
    - [ ] 开发人员工具
        - [ ] 数据源管理
        - [ ] 在线SQL执行
        - [ ] 脚本管理
